• git init: Inicia el repositori i Git crea la carpeta .git. La branca creada és la master.
  • git status ens dona informació
  • touch prova3.html crea un arxiu(comanda linux)
  • git add és per indicar a git quins arxius volem = Posar arxius en el stage area
  • Marcar i desmarcar un arxiu amb add és una acció individual. En canvi pujar un registre ha de ser en bloc.
  • git add -A, t'assegures que el què hagis esborrat desaparegui del commit
  • git rm --cached prova3.html Serveix per fer treure de l'stage area l'arxiu prova3.html
  • git commit -m "Missatge per descriure la tasca realitzada"
